# README – HOPITAL+

##  Nom du projet

**HOPITAL+: Plateforme de prise de rendez-vous médical en ligne**



## Description

HOPITAL+ est une application web conçue pour faciliter la prise de rendez-vous médicaux en Côte d’Ivoire.
Elle permet aux patients de réserver une consultation sans se déplacer, en utilisant une interface simple et accessible.



##  Problématique

En Côte d’Ivoire, les centres de santé font face à plusieurs difficultés :

* longues files d’attente
* mauvaise organisation des rendez-vous
* perte de temps pour les patients
* difficulté d’accès à l’information

Ces problèmes rendent l’accès aux soins plus difficile pour la population.



##   Solution proposée

Le projet HOPITAL+ propose une plateforme web permettant :

* de prendre un rendez-vous médical en ligne
* de réduire les déplacements inutiles
* d’améliorer l’organisation des consultations



##  Objectifs

* Réduire les files d’attente dans les centres de santé
* Améliorer l’accès aux soins
* Moderniser le système de santé
* Faciliter la communication entre patients et structures médicales


##  Cibles

* Patients
* Centres de santé
* Personnel médical


## Structure du site

Le site est composé de 4 pages principales :

### 1. Accueil

* Présentation du projet
* Accès rapide aux services

### 2. Rendez-vous

* Formulaire de prise de rendez-vous
* Saisie des informations du patient

###  3. À propos

* Explication du projet
* Contexte et objectifs

###  4. Nous contacter

* Informations de contact
* Formulaire de message


##  Fonctionnement

###  Version actuelle (prototype)

* L’utilisateur remplit un formulaire
* Les données sont simulées
* Une confirmation est affichée

###  Version future (réelle)

* Les données seront enregistrées dans une base de données
* Les hôpitaux pourront accéder aux rendez-vous
* Le système sera connecté aux structures de santé


## Technologies utilisées

* HTML
* CSS
* JavaScript


##  Perspectives d’évolution

* Création de comptes utilisateurs
* Tableau de bord pour les hôpitaux
* Notifications SMS ou email
* Application mobile
* Intégration avec les systèmes hospitaliers


##  Conclusion

Le projet HOPITAL+ montre comment les technologies web peuvent être utilisées pour résoudre des problèmes concrets dans le domaine de la santé en Côte d’Ivoire.
Il constitue une base pour une future solution numérique capable d’améliorer l’accès aux soins et l’organisation des services médicaux.
